# Sprites GO

Sprites GO is an iOS-first SwiftUI prototype for a kawaii AR virtual pet game.

## Open In Xcode

Open `SpritesGO.xcodeproj`, select an iPhone simulator or device, and run the `SpritesGO` target.

AR camera placement requires a physical iOS device. The app includes a simulator-safe fallback view so the rest of the game can be tested without camera hardware.

## Implemented Prototype

- Pastel SwiftUI home screen with money display and main navigation.
- Settings, Backpack, Shop, Salon, and AR mode screens.
- Local persistence for money, inventory, active sprite, and settings.
- 50 collectible sprites with generated color palettes.
- Starting Flame Fox sprite.
- Consumable and reusable inventory behavior.
- Salon grooming effects with reward payout.
- ARKit horizontal-plane setup with non-AR fallback.
